📖 Грамматика юнита

1. Conditionals — 1st & 2nd
1st (реально): If it rains, we'll stay in. — if + Present, will
2nd (фантазия): If I had more money, I'd travel the world. — if + Past, would
unless = if not: We'll be late unless we hurry.
2. Quantifiers — сколько?

some (утв.) / any (вопр., отриц.) · too much luggage · not enough time

🎧 Аудирование

Planning a trip

Послушай диалог (озвучивается прямо в браузере), потом ответь на вопросы. Слушай сколько угодно раз.

Rosa: I'm so excited! If we save enough, we'll go to Portugal in June.
Jack: Nice! Have you booked the flights yet?
Rosa: Not yet. There aren't many cheap ones left, so we should hurry.
Jack: If I were you, I'd book today. Do you need any help packing?
Rosa: Thanks, but I can do it myself. I've made a list.
Jack: Good idea. Don't take too much luggage — trains have little space.
Rosa: True. I'd recommend travelling light. It's worth seeing the old town too.
Jack: Definitely. Enjoy yourself — and send me a postcard!
